## 1. 产品概述

个人密码安全与管理APP是一款面向个人用户的网络账号密码全生命周期保护工具，旨在解决用户密码管理混乱、密码强度不足、泄露风险不自知等痛点，帮助用户建立安全、高效的密码使用习惯。

- 目标用户：有多个网络账号的普通个人用户，尤其关注信息安全的职场人士和金融用户
- 核心价值：一站式密码安全管理，从存储、评估、生成、更新到泄露检测，构建密码全生命周期防护体系

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 主密码注册 | 管理个人密码库、查看安全报告、使用密码生成器 |

### 2.2 功能模块

1. **密码库主页**：密码总览、分类导航、搜索筛选、安全状态概览
2. **密码管理页**：添加/编辑/删除账号、密码强度评估、自动填充
3. **密码生成器**：一键生成强密码、自定义规则、历史记录
4. **分类管理页**：分类创建/编辑、设置更新周期、到期提醒
5. **泄露检测页**：每日自动扫描、泄露记录查看、预警推送
6. **安全报告页**：月度安全报告、强度分布图、导出加密PDF
7. **设置页**：主密码管理、安全问题、紧急联系人、通知偏好

## 3. 核心流程

### 3.1 密码添加流程
用户点击添加按钮 → 填写网站/应用信息 → 输入或生成密码 → 系统自动评估密码强度 → 若强度不足弹出改进建议 → 用户确认保存 → 密码条目归入相应分类

### 3.2 泄露检测流程
系统每日定时扫描 → 将用户密码哈希与泄露库比对 → 发现匹配 → 推送紧急预警通知 → 用户查看泄露详情 → 系统引导修改密码 → 记录处理状态

### 3.3 密码更新提醒流程
系统检测各分类更新周期 → 发现即将到期的密码 → 推送更新提醒 → 用户点击提醒跳转编辑 → 修改密码 → 记录更新时间 → 重置更新倒计时

### 3.4 主密码重置流程
用户忘记主密码 → 选择验证方式（安全问题/紧急联系人） → 回答安全问题或通过联系人验证 → 验证通过 → 设置新主密码 → 重新加密所有数据

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝黑（#0A0E27）作为主背景，搭配翡翠绿（#00D68F）作为安全/成功色，珊瑚红（#FF4757）作为警告/危险色，琥珀黄（#FFA502）作为中等强度色
- **按钮风格**：圆角胶囊按钮，带微妙的渐变和投影，主操作按钮使用翡翠绿渐变
- **字体**：标题使用 Outfit（几何感、现代科技风），正文使用 DM Sans（清晰易读）
- **布局风格**：左侧固定导航栏 + 右侧内容区，卡片式布局，毛玻璃效果面板
- **图标风格**：线性图标搭配柔和的渐变底色圆角方块，科技安全感
- **整体风格**：暗色赛博朋克安全感，深色背景搭配霓虹色彩点缀，毛玻璃质感卡片，营造安全堡垒般的氛围

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 密码库主页 | 安全状态卡片 | 毛玻璃卡片，左侧数字大号加粗，右侧渐变图标，hover时轻微上浮动画 |

### 4.3 响应式设计

- 桌面优先设计，导航栏在桌面端为左侧固定侧边栏
- 平板端导航栏折叠为图标模式
- 移动端导航栏变为底部标签栏，卡片从多列变为单列
- 触摸优化：按钮最小点击区域44px，列表项左滑操作
